# 硬件操作手册 — DeepBCI 8 通道

## 设备清单

| 物品 | 数量 | 用途 |
|------|------|------|
| DeepBCI 主板 | 1 | 脑电采集（ADS1299 + STM32） |
| USB 数据线 | 1 | 连接电脑 / 树莓派 |
| 电极帽或独立电极 | 8+1+1 | 8 采集 + 1 参考 + 1 地 |
| 导电膏 / 导电胶 | 1 支 | 降低头皮阻抗 |
| 酒精棉片或湿纸巾 | 若干 | 清洁皮肤 |

## 电极位置

使用国际 10-20 系统，覆盖双侧运动皮层：

```
        前面
    Fz
FC3───FCz───FC4     ← 前运动区 (SMA)
    C3───Cz───C4     ← ★ 初级运动皮层 (MI 核心)
    CP3──CPz──CP4    ← 体感反馈区
        后面
```

### 八个通道

| 硬件 Ch | 10-20 位置 | 脑区 | MI 意义 |
|:--:|------|------|------|
| 1 | **FC3** | 左前运动区 | 运动准备 |
| 2 | **C3** | 左中央区 | ★ 右手运动想象 |
| 3 | **Cz** | 中央中线 | 参考/辅助 |
| 4 | **C4** | 右中央区 | ★ 左手运动想象 |
| 5 | **FC4** | 右前运动区 | 运动准备 |
| 6 | **CP3** | 左中央-顶叶 | 体感 |
| 7 | **CPz** | 中线 | 体感参考 |
| 8 | **CP4** | 右中央-顶叶 | 体感 |

### 参考电极与地

| 电极 | 位置 | 说明 |
|------|------|------|
| **REF**（参考） | 左耳垂或左乳突 | 差分放大的参考点 |
| **GND**（地） | 额头正中或右乳突 | 共模噪声抑制 |

> **C3/C4 不能贴错。** 想象左手 → C4（右脑）活跃；想象右手 → C3（左脑）活跃。贴反了信号是反的。

## 如何找到 C3/C4

1. 从鼻根（眉心）到枕外隆凸（后脑勺正中凸起）画一条中线
2. 从左耳前到右耳前画一条横线
3. 两线交点 = **Cz**（中央中线顶点）
4. 从中线往左/右各量 20% 耳前距 = **C3** / **C4**
5. 从 C3/C4 各往前 3cm = **FC3** / **FC4**
6. 从 C3/C4 各往后 3cm = **CP3** / **CP4**

> 简易法：戴电极帽，按帽上标签插电极。10% 误差可接受——采集后会做 CAR 和 EA 对齐。

## 操作步骤

### 1. 连接硬件

```
DeepBCI 主板 ──USB──▶ 电脑
     │
     ├── Ch1~Ch8 → 电极帽对应位置
     ├── REF     → 耳垂夹
     └── GND     → 额头贴片
```

### 2. 给电极打胶

1. 用酒精棉片擦拭每个电极位
2. 每孔注入约豌豆大小导电膏
3. 插入电极，轻轻旋转，确保膏体贴合头皮
4. 参考耳垂同理，地电极贴在额头

### 3. 检查阻抗

```bash
# 打开 OpenBCI GUI（与 DeepBCI 兼容）
# 看每个通道的阻抗值，应 < 10 kΩ
# 绿色 = 好，黄色 = 勉强，红色 = 重新打胶
```

### 4. 验证信号

```bash
python realtime/deepbci_stream.py
# 应该看到 8 条波形，每条约 0~100 μV
# 咬牙 → 所有通道出现高频毛刺
# 眨眼 → 前部通道出现大尖峰
# 闭眼 → 后部通道 α 波明显
```

### 5. 打开在线看板

```bash
python main.py dashboard
# 浏览器 http://localhost:8501
# 左侧：8 通道实时波形
# 右侧：频谱 + 分类结果
```

## 实验范式

### 运动想象标准范式

```
┌──────┐  ┌────────────┐  ┌──────────┐  ┌──────┐
│ 注视  │→ │ 提示（←/→） │→ │ 想象抓握  │→ │ 休息  │
│ 2s   │  │ 1.5s       │  │ 3~4s     │  │ 2~3s │
└──────┘  └────────────┘  └──────────┘  └──────┘
```

1. 屏幕出现箭头（← 左手 / → 右手）
2. 闭眼，想象**缓慢抓握**对应手（不要真的动）
3. 休息期放松，等待下一个 cue

### 录制命令

```bash
python main.py record
# 参数可调: trial 数、休息时长、cue 时长
```

## 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 某通道全是一条直线 | 电极没接触 / 线断了 | 重新插拔电极，检查导电膏 |
| 所有通道跳动很大 | 参考电极或地电极松了 | 重夹耳垂，检查 GND 贴片 |
| 50Hz 嗡嗡干扰 | 交流电源干扰 | 确保陷波器开启（默认开） |
| 频谱全频段能量低 | 阻抗太高 | 重打导电膏、清洁头皮 |
| 运动想象分类准确率低 | 被试疲劳 / 电极偏移 | 每 20 个 trial 休息 2 分钟 |
| EEGNet 分类 100% | 过拟合 / 数据泄漏 | 确认用 LOSO，不是随机 split |

## 阻抗须知

| 阻抗 | 质量 | 建议 |
|------|------|------|
| < 5 kΩ | 优秀 | 直接用 |
| 5~10 kΩ | 良好 | 可用 |
| 10~20 kΩ | 勉强 | 重打导电膏 |
| > 20 kΩ | 差 | 必须重新来 |

> 头发干燥、油脂、发胶都会升高阻抗。务必先擦酒精。

---

## 安全

- 设备由 USB 供电（5V），**不接市电**
- 导电膏只涂头皮，不进入眼睛
- 每次使用后清洗电极帽
- 被试头痛、不适立即停止
