# 面向 8 通道运动想象 EEG 的黎曼几何表征学习与跨被试解码方法研究

> **项目**: XH-202610 基于运动想象的脑－机交互算法研究  
> **周期**: 2025 年 5 月 – 2025 年 11 月  
> **团队**: 项目组协作；当前阶段由负责人主导完成算法调研、基线复现与技术路线验证  
> **硬件**: 8 通道运动皮层 EEG（FC3, C3, Cz, C4, FC4, CP3, CPz, CP4）@ 250 Hz  
> **数据集**: PhysioNet MI (30 subjects, binary) + BCI Competition IV 2a (9 subjects, 4-class)  
> **当前项目最佳基线**: EEG Conformer + EA @ 63.93%（PhysioNet MI 8ch binary LOSO）

---

## 一、研究背景

运动想象脑机接口（Motor Imagery BCI, MI-BCI）通过分析用户在想象肢体运动时产生的脑电信号，实现无外部肌肉动作的人机交互控制。该方向在康复训练、辅助控制和智能交互等场景中具有较高应用价值。

然而，MI-EEG 信号具有低信噪比、强非平稳性和显著个体差异。尤其在跨被试场景下，不同被试之间的脑电分布差异明显，导致模型在训练被试上表现较好，但迁移到新被试时性能下降。对于实际 BCI 系统而言，减少新用户校准数据、提升跨被试泛化能力，是当前算法研究的关键问题。

本项目现有条件为 8 通道运动皮层 EEG 数据，主要覆盖 FC3、C3、Cz、C4、FC4、CP3、CPz、CP4 等运动相关脑区通道。现有实验中，基于 EA 对齐的 EEG Conformer 在 binary LOSO 评估下取得约 63.93% 准确率，基于 Tangent Space + LDA + EA 的传统黎曼方法取得约 60.44% 准确率。**二者差距仅 3.49pp**，说明协方差结构已经包含了大量与运动想象分类相关的信息——深度网络从原始 EEG 中学到的大部分内容，其实是在隐式地逼近协方差结构。

因此，本研究不再单纯追求将 CNN、Transformer、Mamba、KAN 等深度结构进行架构替换，而是将研究重点放在更适合 EEG 信号结构的表征空间上：**在 SPD 协方差流形上进行深度表征学习**，并结合自监督预训练提高跨被试泛化能力。

---

## 二、核心问题

当前 MI-EEG 解码方法主要面临以下问题：

**问题一：原始 EEG 输入效率低。** 原始 EEG 直接输入深度网络时，模型需要同时学习时域滤波、空间相关性和分类判别边界。在样本量较小、被试差异较大的情况下，深度模型容易过拟合训练被试。协方差矩阵是 EEG 空间结构的自然压缩表示，但现有深度方法大多绕开了这一表示。

**问题二：传统黎曼方法表达能力有限。** 传统黎曼方法（Tangent Space + LDA、MDM 等）能够利用 EEG 协方差矩阵的几何结构，具有较好的稳定性和泛化性，但多数方法最终仍将 SPD 矩阵投影到切空间后使用线性分类器，非线性表达能力不足。

**问题三：8 通道低通道配置研究不足。** 现有大量 EEG 深度学习研究集中在 22 通道或更多通道的公开数据集（BCI Competition IV 2a、BNCI 系列）上，对于 8 通道运动皮层低通道配置的专门研究较少。8 通道系统更接近实际便携式 BCI 硬件（如本项目使用的 DeepBCI），但也带来了信息量不足和空间分辨率下降的挑战。

**问题四：跨被试少样本适配困难。** 跨被试场景下，新被试可用标注数据通常较少。单纯依赖全监督训练难以充分利用大量无标签 EEG 片段，需要引入自监督学习来提高模型在少样本适配场景下的表现。

---

## 三、研究目标

本研究拟构建一种面向 8 通道运动想象 EEG 的**黎曼几何深度表征方法**，在 EA 对齐和 SPD 流形建模的基础上，提高跨被试运动想象分类性能。

具体目标：

1. 建立 8 通道 MI-EEG 的标准化跨被试评估流程，包括预处理、EA 对齐、协方差估计、LOSO 评估和统计检验。
2. 构建基于 SPD 流形的深度分类模型（SPDNet / Tensor-CSPNet / TSMNet 等），比较其与 EEGNet、EEG Conformer、Tangent Space + LDA 等方法的性能差异。
3. 引入运动皮层通道拓扑先验，使模型能够更好地利用左右半球及中央区通道之间的空间关系。
4. 探索 SPD 表征上的自监督预训练方法（对比学习 / 掩码重建），提升模型在跨被试和少样本场景下的泛化能力。
5. 在 PhysioNet MI 和 BCI Competition IV 2a 等公开数据集上完成对比实验、消融实验和可视化分析。

## 四、总体技术路线

```
8ch 原始 EEG
     ↓
预处理（带通滤波 8–30 Hz + 陷波 50 Hz + 试次截取）
     ↓
EA 对齐（跨被试协方差对齐）
     ↓
频段划分（可选：mu 8–13 Hz, beta 13–30 Hz, full 8–30 Hz）
     ↓
协方差矩阵计算（8×8 SPD 矩阵）
     ↓
┌─────────────────────────────────────┐
│  SPD 深度网络（BiMap → ReEig → LogEig）│
│  + 运动皮层拓扑先验                     │
│  + 自监督预训练（对比学习 / 掩码重建）     │
└─────────────────────────────────────┘
     ↓
分类器 → LOSO 跨被试评估
```

**与传统黎曼方法的关键区别**：
- 传统：SPD 矩阵 → 切空间投影（线性化）→ LDA（线性分类）
- 本文：SPD 矩阵 → SPDNet 流形上的深度非线性变换 → LogEig → 分类

**与原始 EEG 深度方法的关键区别**：
- 原始 EEG DL：模型隐式学习协方差结构（浪费大量容量）
- 本文：显式计算协方差 → 直接在正确的几何空间建模

## 五、拟采用方法

### 5.1 基线方法

为保证实验结果可信，建立以下基线：

| 类别 | 方法 | 当前已知结果 (binary LOSO) |
|------|------|--------------------------|
| 传统 ML | CSP + LDA | — |
| 传统 ML | FBCSP + LDA | — |
| 传统黎曼 | EA + Tangent Space + LDA | 60.44% |
| 传统黎曼 | EA + MDM / FgMDM | 56.22% / 59.18% |
| 深度学习 | EEGNet | 51.93% (no EA) / 58.00% (+EA) |
| 深度学习 | EEG Conformer + EA | 63.93% |
| 深度学习 | EEG-TCNet + EA | 63.41% |

### 5.2 黎曼深度学习模型

核心模型以 8×8 SPD 协方差矩阵为输入，使用 BiMap、ReEig、LogEig 等结构完成流形上的特征变换：

```
原始 EEG → 频段滤波 → EA 对齐 → 协方差估计 → SPD 深度网络 → 分类器
```

**基础 SPD 模型**（使用 spd-learn 库）：

| 模型 | 特点 |
|------|------|
| SPDNet | BiMap → ReEig → LogEig，基础几何 DL |
| TSMNet | Tangent Space Mapping + SPD BatchNorm |
| Tensor-CSPNet | 多频段时间分割 + BiMap，时空频联合 |
| PhaseSPDNet | 相位信息（Hilbert 变换）+ SPDNet |

若采用多频段输入，则每个频段分别计算协方差矩阵，再通过多分支结构进行融合。

### 5.3 运动皮层拓扑先验

8 通道电极具有明确的运动皮层空间分布：

```
左半球: FC3, C3, CP3
中线:   Cz, CPz
右半球: FC4, C4, CP4
```

该先验可通过以下方式引入：
- 在协方差矩阵中保留通道顺序，使矩阵结构对应运动皮层空间关系
- 设计通道邻接 mask，突出左右运动区和中央区之间的功能连接
- 在多分支模型中分别建模左侧、右侧和中线通道，再进行融合

目标不是强行增加模型复杂度，而是让模型在低通道条件下更充分地利用有限通道的解剖信息。

### 5.4 SPD 表征上的自监督预训练（第二阶段增强）

自监督预训练作为**第二阶段增强模块**开展，不作为第一阶段主模型的必要组成部分。优先探索**对比学习方法**：对同一 trial 构造两种增强视图（轻微时间扰动、幅值缩放、通道 dropout、频段 mask 等），使模型学习在扰动下保持稳定的跨被试表征。

掩码重建作为扩展实验保留：若对比学习已能带来稳定提升，再进一步尝试遮蔽部分通道或频段，要求模型重建对应的协方差结构。

预训练完成后，将编码器迁移到分类任务中，分别测试全量微调、冻结 backbone 只训练分类头、以及 5-shot / 10-shot / 20-shot 少样本微调等设置。

### 5.5 备选与未来方向（不在当前阶段主攻）

| 方向 | 状态 | 理由 |
|------|------|------|
| Mamba/SSM | 备选对比实验 | 已有多个 MI-Mamba 工作，原创性不足；可作为时间建模的对比方案 |
| KAN | 暂不纳入 | 训练慢（~10× MLP），对实验周期不友好；可解释性亮点有价值但非主线 |
| 扩散模型数据增强 | 暂不纳入 | 作为数据增强手段可与 SPD 框架结合，但不应作为主创新 |
| SNN | 暂不纳入 | 训练困难，生态不成熟，精度天花板较低 |
| 源空间解码 | 未来工作 | 8 通道源定位高度病态，需先验证传感器空间方案有效性 |
| 预测编码/世界模型 | 未来工作 | 理论上限高但实现风险大，待基础方案稳定后探索 |

## 六、创新点

本研究的创新点不在于简单替换网络结构，而在于**针对 8 通道运动皮层 EEG 的信号特点，构建更适合跨被试泛化的几何表征学习框架**。

**创新点一**：面向 8 通道运动皮层 MI-EEG 场景，构建基于 SPD 流形的深度解码模型。相比标准 22 通道或全通道 EEG，8 通道系统更接近便携式 BCI 硬件部署需求，但也面临空间信息不足和跨被试泛化困难的问题。现有 SPD 黎曼深度学习方法虽已在 MI-EEG 中取得进展，但针对 DeepBCI 式 8 通道运动皮层蒙太奇、PhysioNet binary LOSO 评估和实际硬件验证的系统研究仍较少，具有明确的差异化研究空间。

**创新点二**：将 EA 对齐与 SPD 协方差深度表征学习结合，形成面向跨被试 MI-EEG 解码的几何建模流程。EA 通过对齐不同被试 EEG trial 的协方差统计特征，降低跨被试分布偏移；SPD 深度网络进一步在协方差流形上进行非线性表征学习。二者结合能够同时利用传统黎曼方法的稳定性和深度模型的表达能力。

**创新点三**：引入运动皮层通道拓扑先验，在低通道条件下增强模型对左右半球运动区差异的利用能力。8 通道恰好覆盖运动皮层的核心区域（双侧 M1/S1 + SMA），这一解剖对应关系未被现有 SPD 方法显式利用。

**创新点四**：探索 SPD 表征上的自监督预训练。现有黎曼 DL 方法均为全监督训练，尚未有工作在 SPD 流形上结合对比学习或掩码重建进行预训练。该方向若验证有效，可为低通道 MI-EEG 的跨被试泛化提供新的训练范式。

## 七、实验设计

### 7.1 评估协议

采用 **LOSO（Leave-One-Subject-Out）跨被试评估**。每次选择一个被试作为测试集，其余被试作为训练集，循环完成所有被试测试，最终报告平均准确率 ± 标准差，并进行配对 t 检验或 Wilcoxon 符号秩检验。

### 7.2 主要实验

| 实验 | 目的 | 对比项 |
|------|------|--------|
| **基线对比** | 验证本文方法有效性 | EEGNet, EEG Conformer, Tangent Space + LDA, SPDNet 等 |
| **EA 消融** | 验证 EA 对齐的贡献 | 有无 EA 的各方法对比 |
| **SPD 模型消融** | 比较不同 SPD 架构 | Tangent Space + LDA, SPDNet, TSMNet, Tensor-CSPNet |
| **频段消融** | 验证频段划分贡献 | 单频段 vs 多频段 vs 滤波器组协方差 |
| **SSL 消融** | 验证预训练贡献 | 无预训练 vs 对比学习 vs 掩码重建 vs 联合预训练 |
| **少样本实验** | 验证少样本适配能力 | 5-shot, 10-shot, 20-shot, 全量微调 |
| **可解释性** | 验证模型学到合理模式 | 协方差响应、通道贡献、频段贡献、特征可视化 |

### 7.3 预期结果

| 目标层次 | 描述 | 参考值 |
|----------|------|--------|
| **保底目标** | 稳定超过 Tangent Space + LDA + EA | >60.44% |
| **合格目标** | 超过当前 EEG Conformer + EA | >63.93% |
| **理想目标** | 在严格 LOSO 下达到 66–70% | 66–70% |
| **冲刺目标** | 在少样本场景体现 SSL 预训练优势 | ≤20-shot 下优于全监督 |

> 若主模型未显著超过 EEG Conformer + EA，则以低通道 SPD 建模流程、消融分析和少样本适配结果作为主要输出。研究贡献不单纯依赖准确率数值。

## 八、阶段计划

本项目采用"负责人先行验证、团队后续扩展"的推进方式。当前阶段由负责人主导完成技术路线验证和最小实验闭环；后续团队成员加入后，再开展多分支实验和系统集成。

| 阶段 | 周期 | 人力状态 | 内容 |
|------|------|----------|------|
| **第一阶段** | 1–2 周 | 负责人主导 | **基线复现与流程标准化**。复现 EEGNet、EEG Conformer、EA + Tangent Space + LDA，确认 LOSO 评估流程可靠，建立自动化实验脚本。 |
| **第二阶段** | 2–3 周 | 负责人主导 | **SPD 深度学习模型初步实现**。基于 spd-learn 库或自实现 SPDNet，完成 8ch 输入适配，建立首个 8ch SPD DL 基线。 |
| **第三阶段** | 1–2 周 | 团队协作 | **多频段协方差与运动皮层拓扑先验**。引入多频段协方差矩阵和通道拓扑先验，完成模型结构消融。 |
| **第四阶段** | 2–3 周 | 团队协作 | **SSL 预训练实验（选做增强）**。优先实现 SPD 表征上的对比学习预训练，掩码重建作为扩展。测试全量 LOSO 和少样本适配场景。 |
| **第五阶段** | 2 周 | 团队协作 | **实验整理与论文**。完成所有对比实验、消融实验、可视化、统计检验和论文初稿。 |

任务分层一览：

| 层级 | 内容 |
|------|------|
| **必做** | EA + Tangent Space、EEGNet / Conformer 复现、SPDNet / TSMNet、LOSO 评估 |
| **强做** | 多频段协方差、运动皮层拓扑先验 |
| **选做** | SSL 对比学习预训练 |
| **不做或写未来工作** | 掩码重建、PhaseSPDNet、源空间解码、Mamba、KAN、SNN、扩散模型 |

## 九、风险与应对

| 风险 | 应对 |
|------|------|
| **SPD 深度模型相比 Tangent Space + LDA 提升有限** | 保底假设：即使 SPDNet 本身提升不大，频段协方差 + 拓扑先验 + SSL 的组合仍有提升空间。保留传统黎曼方法作为强基线，确保即使"SPD DL"不够好，研究框架仍有输出。 |
| **SSL 预训练数据规模不足** | PhysioNet 30 被试 × 约 100 trials ≈ 3000 samples 偏少。先在无 SSL 条件下建立可靠主模型，SSL 作为增强模块。必要时引入 BCI IV 2a、BNCI 系列等公开数据扩充预训练集。 |
| **模型复杂度过高导致小样本过拟合** | 优先使用轻量化 SPD 网络（如基础 SPDNet），控制参数规模（8×8 协方差 → BiMap 降维到 4×4 或更低）。通过 LOSO 和消融实验判断泛化能力。 |
| **性能提升不足以支撑论文创新** | 将贡献从"单纯追求准确率"转向"8 通道低通道 MI-EEG 的几何表征框架、跨被试评估基准和少样本适配分析"。在 8ch 场景下建立方法对比基准本身就有学术价值。 |

## 十、与现有方向调研的关系

本方案是在前期方向调研（见 `docs/research_directions_2025.md`）基础上，经过删繁就简后确定的主线方案。调研阶段考察的 Mamba、KAN、扩散模型、SNN、源空间解码等方向，均降级为备选或未来工作，不纳入当前阶段的实验计划。

选择"黎曼几何表征学习"作为主线的核心逻辑：
1. 协方差特征已证明其有效性（60.44% vs 原始 EEG 51.93%）
2. SPD 流形深度学习有成熟基础设施（spd-learn 库），但对 DeepBCI 式 8ch 运动皮层蒙太奇 + PhysioNet binary LOSO 的组合研究仍不足
3. 与 EA 对齐有自然的几何统一解释
4. 不是"换架构"，而是"换表征空间"——创新叙事更清晰
